### Upload changes
Go into directory where the repo is located, e.g.:
```
cd ~/Documents/GitHub/Snow-e
```
Get updates done by other collaborators:
```
git fetch origin              
git status 
git pull
```

Upload your changes:
```
git add .
git commit -m “your text, e.g. updated function A”
git push
```
